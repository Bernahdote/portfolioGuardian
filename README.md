# portfolioGuardian
Notifies users if they need to do something with their portfolio.

## Setup openai key (zsh):
- Copy the key.
- `echo 'export OPENAI_API_KEY="sk-yourkeyhere"' >> ~/.bash_profile`
- `source ~/.bash_profile`


## Setup openai key (bash):
- Copy the key.
- `echo 'export OPENAI_API_KEY="sk-yourkeyhere"' >> ~/.bash_profile`
- `source ~/.bash_profile`
