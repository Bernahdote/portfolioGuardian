# portfolioGuardian
Notifies users if they need to do something with their portfolio.

## Setup openai key (zsh):
- Copy the key.
- `echo 'export OPENAI_API_KEY="sk-yourkeyhere"' >> ~/.zshrc`
- `source ~/.zshrc`


## Setup openai key (bash):
- Copy the key.
- `echo 'export OPENAI_API_KEY="sk-yourkeyhere"' >> ~/.zshrc`
- `source ~/.bash_profile`


## Setup email SMTP password (zsh):
- Ask Teo for his password.
- `echo 'export SMTP_PASSWORD="paste password"' >> ~/.bash_profile`
- `source ~/.zshrc`


## Setup email SMTP password (bash):
- Ask Teo for his password.
- `echo 'export SMTP_PASSWORD="paste password"' >> ~/.bash_profile`
